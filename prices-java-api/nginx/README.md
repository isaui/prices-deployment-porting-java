# Nginx Configuration

This directory contains **initial/static** nginx configurations that are versioned in Git.

## Directory Structure

```
nginx/
├── conf.d/
│   └── default.conf    # Deployment API config + catch-all
└── README.md
```

## How it works

Nginx container mounts TWO directories:

1. **`./nginx/conf.d`** → `/etc/nginx/conf.d/static/` (initial configs, versioned)
2. **`./docker-vol/nginx/conf.d`** → `/etc/nginx/conf.d/dynamic/` (dynamic project configs, generated)

All `*.conf` files from both directories are loaded by nginx.

## Initial Configs

- `default.conf`: Routes `api.prices.dev` to deployment API + catch-all 404

## Dynamic Configs

Generated by deployment pipeline at `/var/prices/nginx/conf.d/{slug}.conf`
